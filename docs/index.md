# SpecGantry

AI-powered SDLC pipeline for Claude Code with enforced phase gates, from ideation to deployment.

## Overview

SpecGantry is a comprehensive AI-assisted software development lifecycle (SDLC) framework designed to work seamlessly with Claude Code. It provides structured workflows, enforced phase gates, and intelligent agents to guide your development process from initial specification through to production deployment.

## Features

- **Phase-Gated Workflows**: Enforced gates between ideation, design, implementation, testing, and deployment phases
- **AI-Powered Agents**: Specialized agents for different aspects of the development lifecycle
- **Claude Code Integration**: Fully integrated with Claude Code as a plugin
- **Configuration-Driven**: Easy-to-customize pipeline configuration
- **Best Practices**: Built-in patterns for code generation, testing, and deployment

## Getting Started

### Install as Claude Code Plugin

Add SpecGantry directly to Claude Code:

```
https://github.com/specgantry/specgantry.github.io
```

Or search for "spec-gantry" in Claude Code's plugin marketplace.

### Project Structure

- `/agents` - Specialized agents for different development phases
- `/skills` - Reusable skills and utilities
- `/config` - Configuration templates and examples
- `/docs` - Documentation and guides
