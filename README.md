# SpecGantry

AI-powered SDLC pipeline for Claude Code with enforced phase gates, from ideation to deployment.

## About This Repository

SpecGantry serves dual purposes:

1. **Claude Code Plugin** - Install directly in Claude Code for AI-assisted development workflows
2. **GitHub Pages Site** - Documentation and guides at [specgantry.github.io](https://specgantry.github.io)

## Quick Start

### Install as Claude Code Plugin

Open Claude Code and add this repository as a plugin:

```
https://github.com/specgantry/specgantry.github.io
```

### View Documentation

Visit [specgantry.github.io](https://specgantry.github.io) for full documentation, guides, and examples.

## Repository Structure

```
.
├── .claude-plugin/          # Claude Code plugin configuration
├── agents/                  # Specialized agents for development phases
├── skills/                  # Reusable skills and utilities
├── config/                  # Configuration templates and examples
├── docs/                    # GitHub Pages documentation (published to web)
├── CONTRIBUTING.md          # Contribution guidelines
├── SECURITY.md              # Security policy
└── LICENSE                  # Apache 2.0 License
```

## Features

- **Phase-Gated Workflows** - Enforced gates between development phases
- **AI-Powered Agents** - Specialized agents for different aspects of SDLC
- **Claude Code Integration** - Full integration with Claude Code
- **Configuration-Driven** - Easy-to-customize pipeline configuration
- **Best Practices** - Built-in patterns for code generation, testing, and deployment
