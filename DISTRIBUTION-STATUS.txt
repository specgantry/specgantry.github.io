RELEASE WORKFLOW
================

To release SpecGantry:

1. Update version in 3 files:
   - package.json
   - .claude-plugin/plugin.json
   - marketplace.json

2. Commit: git commit -m "Release vX.Y.Z"

3. Tag: git tag -a vX.Y.Z -m "Release vX.Y.Z"

4. Push: 
   - git push origin main
   - git push origin vX.Y.Z

5. GitHub Actions automatically:
   - Creates spec-gantry.zip
   - Publishes GitHub Release
   - Makes available for installation

Total time: ~5 minutes
Manual steps: 4 git commands

================================================================================
INSTALLATION FOR USERS
======================

After release, users can install via:

  claude plugin install https://github.com/specgantry/specgantry.github.io

Or when registered with marketplace:

  claude plugin install spec-gantry

Or for local development:

  claude plugin install /path/to/spec-gantry

All three methods are supported.
